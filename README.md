## 🌟 Overview

CodeFlux AI is a futuristic full-stack AI SaaS platform that helps developers analyze, optimize, secure, and understand code instantly using AI.

Paste your code and get:
- AI Code Explanation
- Bug Detection
- Security Analysis
- Complexity Analysis
- Refactoring Suggestions
- Documentation Generation
- README Generation
- AI Code Chat
- Performance Optimization
